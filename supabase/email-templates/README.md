# Supabase Email Templates

Custom branded email templates for Illini Market. Bilingual (中英).

## How to apply

1. Open Supabase Dashboard → Authentication → Email Templates
2. For each template:
   - `Confirm signup` → paste contents of `confirm-signup.html`
   - `Reset password` → paste contents of `reset-password.html`
3. Keep the `{{ .ConfirmationURL }}` token as-is — Supabase substitutes it.

## Preview

Open the .html files directly in a browser to preview.
