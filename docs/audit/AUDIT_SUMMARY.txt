  - Low contrast text (WCAG AA failures)
  - Inconsistent loading/error states
  - Missing accessibility attributes

================================================================================
TOP 5 CRITICAL ISSUES
================================================================================

1. HARDCODED ENGLISH ERROR MESSAGE (App.vue:36)
   - "No network connection" not translated
   - Chinese users see English
   - Fix: Add i18n key 'error.noNetwork'

2. MISSING ALT ATTRIBUTES ON IMAGES (24 instances)
   - Product images, avatars, message images
   - Fails WCAG 2.1 Level A
   - Fix: Add alt="..." to all <image> tags

3. HARDCODED ERROR MESSAGES IN COMPOSABLES
   - "Failed to load plaza", "Failed to load messages"
   - Chinese users see English
   - Fix: Add i18n keys for all errors

4. LOGIN FORM EMAIL INPUT TYPE
   - Uses type="text" instead of type="email"
   - Mobile keyboards don't show @ symbol
   - Fix: Change to type="email"

5. PASSWORD AUTOCOMPLETE MISSING
   - No autocomplete="current-password" or "new-password"
   - Password managers can't autofill
   - Fix: Add autocomplete attribute

================================================================================
CONTRAST ISSUES (WCAG AA Failures)
================================================================================

- #999 gray on white (filter labels, empty state text)
- #ccc gray on white (price range dash)
- #bbb gray on white (helper text)
- #8e8e93 gray on white (seller stats labels)

Fix: Change all to #666 or darker

================================================================================
ACCESSIBILITY GAPS
================================================================================

Images:
  - 24 <image> tags missing alt attributes
  - Affects: products, avatars, messages, thumbnails

Forms:
  - Email input: type="text" (should be type="email")
  - Password: missing autocomplete attributes
  - No <label> elements for inputs
  - Character counters not labeled for screen readers

Touch Targets:
  - Back buttons: 32px (should be 44px minimum)
  - Close buttons: 32px (should be 44px minimum)
  - Filter buttons: 32px (should be 44px minimum)

Keyboard Navigation:
  - Custom buttons don't respond to Enter/Space
  - No focus indicators visible
  - No keyboard trap prevention

Screen Readers:
  - No aria-live on toasts
  - No aria-label on icon buttons
  - No aria-hidden on decorative elements
  - No role="button" on clickable divs

================================================================================
I18N COVERAGE GAPS
================================================================================

Hardcoded English Strings:
  - "No network connection" (App.vue:36)
  - "Failed to load plaza" (usePlaza.ts)
  - "Failed to load messages" (useMessages.ts)
  - "Failed, please try again" (useFavorites.ts)
  - Time ago strings: "5m ago", "2h ago", "3d ago" (formatTime)
  - Currency symbol: $ (hardcoded in multiple places)

Language Conditionals (Fragile):
  - lang.value === 'zh' ? spot.zh : spot.en (5+ files)
  - Should use i18n keys instead

Date/Number Formatting:
  - formatTime() returns English strings
  - formatPrice() hardcodes $ symbol
  - No toLocaleString() usage

UX CONSISTENCY GAPS
================================================================================

Loading States:
  - Inconsistent across pages
  - Detail page: NO loading state
  - Seller page: HAS loading state
  - Home page: HAS loading state

Error States:

Confirm Dialogs:
  - Some use i18n, some hardcoded

Pull-to-Refresh:
