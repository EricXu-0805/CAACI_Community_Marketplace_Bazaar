# CAACI Community Marketplace Bazaar

> 面向华人社区的同城二手交易 + 社区互动平台

## 项目定位

**闲鱼 + 小红书** 混合模式，服务于华协 (CAACI) 及本地华人社区。

- 初期：**信息发布平台**（不做平台内交易，鼓励线下自行成交）
- 长期：打造同城华人生活服务生态

## 核心功能模块

| 模块 | 描述 |
|------|------|
| **二手交易** | 发布/浏览二手物品（含转租），支持搜索 + 多维度 filter |
| **社区论坛** | 顶部官方便民信息 + 用户发帖区，帖子可评论回复 |
| **私信系统** | 用户间一对一私聊 |
| **个人中心** | 用户资料、华协会员卡认证、历史发帖、会员等级 |

## 多端部署

| 优先级 | 平台 | 说明 |
|--------|------|------|
| P0 | 微信小程序 | 华人用户主阵地，试运行首选 |
| P0 | Web 网站 | 手机端 ≈ App 布局，桌面端 Tab 放顶部导航 |
| P1 | iOS App | 达到一定规模后上架 Apple Store |
